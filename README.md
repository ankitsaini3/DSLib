# 📚 Header-Only C++ Data Structures Library

A modern, template-based, header-only C++ library implementing commonly used data structures. Designed to be lightweight, fast, and easy to integrate into any project with zero dependencies.

---

## ⚙️ Features

- ✅ Header-only design — just `#include` and go!
- ✅ Fully templated classes for type safety and flexibility
- ✅ STL-like naming and conventions
- ✅ Clean and consistent code structure
- ✅ Cross-platform and dependency-free
- ✅ Extensible for custom data structures

---

## 🧱 Implemented Data Structures

| STL Category               | Structures Implemented                                                  |
| -------------------------- | ----------------------------------------------------------------------- |
| **Sequence Containers**    | `vector`, `deque`, `forward_list`                                       |
| **Container Adapters**     | `stack`, `queue`                                                        |
| **Pseudo Containers**      | `string`                                                                |
